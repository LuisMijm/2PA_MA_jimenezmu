Retrieves data from a database query and stores it in data structures.

This function is used to retrieve data from a database query and store it in data structures. It handles both column names and the actual data rows. Column names are stored in the settings.db_ColNames array, and data rows are stored in the settings.db_table_info structure.

Parameters

argvAn array of strings containing the data for each column.
azcolnameAn array of strings containing the column names.

Returns 0 if the operation is successful.
